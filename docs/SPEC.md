# Especificación del Sistema

## Alcance

El proyecto consiste en un Sistema de Gestión de Eventos y Entradas tipo Ticketek.

El sistema estará compuesto por:

- Backend en Go.
- Frontend en React.
- Base de datos MySQL.
- ORM GORM.
- Autenticación con JWT.
- Roles Cliente y Administrador.

## Etapas del proyecto

La primera etapa estará enfocada en alcanzar la regularidad mediante el flujo completo del cliente, incluyendo registro, login, consulta de eventos, compra de entradas, visualización de entradas propias, cancelación y transferencia.

En una etapa posterior se incorporarán funcionalidades de administrador, Docker, bonus y el objetivo de cobertura de tests del 80%.
