# TicketApp

Sistema de Gestión de Eventos y Entradas desarrollado para la materia Desarrollo de Software 2026.

## Descripción

TicketApp permite a los usuarios explorar eventos, comprar entradas, ver sus entradas adquiridas, cancelar compras y transferir entradas a otros usuarios.

El sistema estará compuesto por:

- Backend en Go
- Frontend en React
- Base de datos MySQL
- ORM GORM
- Autenticación con JWT
